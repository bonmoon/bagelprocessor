User Guide

Use the same file directly in the browser. Every module supports upload, preview, and export without exposing any customer-specific labels in the interface.

1

Upload the source workbook

Each module reads the relevant sheet automatically. If a category or product sheet is available, the tool will prioritize it.

2

Preview before export

Use the preview action to verify matched rows, combined category paths, and cleaned columns before downloading a new workbook.

3

Filter multiple category nodes

In the category filter module, select one or many category values from the list. Rows are kept when any selected value appears in the category hierarchy.

4

Apply hierarchy updates

Optionally add a new top-level category. The output includes a Combined Category column such as H *** A *** B *** C.
